# Markdown 数学笔记格式规范

本文档定义了数学笔记 Markdown 文件的标准格式，确保后续 HTML 生成的一致性和质量。

## 1. 基本原则

- 使用 **标准 Markdown** 语法，不使用扩展语法
- 使用 **标准 LaTeX** 命令（KaTeX 兼容）
- 结构标记使用 **加粗** 格式
- 每个结构块之间空一行

## 2. 标题层级

```markdown
# 章节标题

## 一、一级节标题

### 1.1 二级节标题

#### 1.1.1 三级节标题（可选）
```

**规则**：
- 一级标题 `#` 只用于文档标题
- 二级标题 `##` 用于主要章节
- 三级标题 `###` 用于子章节
- 标题后不加 ID、class 等扩展语法

## 3. 定义格式

```markdown
**定义 1.1（函数的连续性）** 设函数 $f(x)$ 在点 $x_0$ 的某邻域内有定义。若

$$\lim_{x \to x_0} f(x) = f(x_0)$$

则称 $f(x)$ 在点 $x_0$ 处**连续**。

**注** 连续性的三个条件：...
```

**规则**：
- 使用 `**定义 X.X（名称）**` 格式
- 定义内容紧跟在标题后
- 使用 `**注**` 添加注释（可选）

## 4. 定理格式

```markdown
**定理 1.1（介值定理）** 设 $f(x)$ 在 $[a,b]$ 上连续，且 $f(a) \neq f(b)$，则对于 $f(a)$ 与 $f(b)$ 之间的任意值 $c$，存在 $\xi \in (a,b)$，使得 $f(\xi) = c$。

**证明** 由连续性...

**证毕** □
```

**规则**：
- 使用 `**定理 X.X（名称）**` 格式
- 证明使用 `**证明**` 开头
- 结尾使用 `**证毕** □`

## 5. 例题格式

```markdown
**例 1.1** 求下列极限：

（1）$\displaystyle\lim_{n \to \infty} \frac{1}{n}$

（2）$\displaystyle\lim_{n \to \infty} \frac{n}{n+1}$

**解** （1）由定义，对于任意 $\epsilon > 0$，...

**注** 本题的关键是...（可选）
```

**规则**：
- 使用 `**例 X.X**` 格式
- 小题使用 `（1）`、`（2）` 编号
- 解答使用 `**解**` 开头

## 6. 列表格式

### 有序列表

```markdown
1. 第一项内容；
2. 第二项内容；
3. 第三项内容。
```

### 无序列表

```markdown
- 要点一
- 要点二
- 要点三
```

**规则**：
- 列表项之间不空行
- 有序列表以分号结尾，最后一项以句号结尾
- 列表前后各空一行

## 7. 数学公式

### 行内公式

```markdown
设函数 $f(x) = x^2 + 1$，则 $f'(x) = 2x$。
```

### 独立公式

```markdown
由泰勒公式：

$$f(x) = f(a) + f'(a)(x-a) + \frac{f''(a)}{2!}(x-a)^2 + \cdots$$
```

**规则**：
- 行内公式使用 `$...$`
- 独立公式使用 `$$...$$`，单独成行
- 公式前后各空一行（独立公式）
- 使用标准 LaTeX 命令

### 常用 LaTeX 命令对照

| 含义 | 标准命令 | 示例 |
|-----|---------|-----|
| 极限 | `\lim_{x \to a}` | $\lim_{x \to a}$ |
| 分数 | `\dfrac{a}{b}` | $\dfrac{a}{b}$ |
| 求和 | `\sum_{i=1}^{n}` | $\sum_{i=1}^{n}$ |
| 积分 | `\int_{a}^{b}` | $\int_{a}^{b}$ |
| 希腊字母 | `\alpha, \beta, \gamma` | $\alpha, \beta, \gamma$ |
| 箭头 | `\to, \Rightarrow` | $\to, \Rightarrow$ |
| 不等式 | `\geq, \leq, \neq` | $\geq, \leq, \neq$ |

## 8. 特殊内容块

### 思考题

```markdown

```

### 作业

```markdown
**作业**

```

### 警告/注意

```markdown
**注意** 使用洛必达法则前，必须验证是否满足条件。
```

## 9. 禁止使用的语法

| 禁止 | 原因 | 替代方案 |
|-----|------|---------|
| `{#id .class}` | Pandoc 扩展 | 不使用 |
| `[文本]{style="..."}` | Pandoc 扩展 | 使用 `**加粗**` |
| `\rr`, `\ep` 等自定义命令 | KaTeX 不支持 | 使用标准命令 |
| HTML 标签 | 不纯 | 使用 Markdown 语法 |

## 10. 完整示例

```markdown
# 第一章 函数与极限

## 一、函数的概念

**定义 1.1（函数）** 设 $D$ 是一个非空数集。如果存在一个对应法则 $f$，使得对于 $D$ 中的每一个数 $x$，都有唯一确定的数 $y$ 与之对应，则称 $f$ 为定义在 $D$ 上的一个**函数**，记作

$$y = f(x), \quad x \in D$$

**注** 函数有两个核心要素：定义域和对应法则。

**例 1.1** 求函数 $f(x) = \sqrt{x-1} + \dfrac{1}{x-2}$ 的定义域。

**解** 要使函数有意义，需要：

1. $x - 1 \geq 0$，即 $x \geq 1$；
2. $x - 2 \neq 0$，即 $x \neq 2$。

所以定义域为 $D = [1, 2) \cup (2, +\infty)$。

**定理 1.1（有界性定理）** 闭区间上的连续函数必有界。

**证明** 略。

**证毕** □

## 二、极限的概念

（后续内容...）
```

- [ ] 数学公式使用标准 LaTeX 命令
